KeyPress OSD: Keyboard shortcuts

All shortcut keys are global, available in any application. These can be customized, enabled or disabled at: Main menu > Preferences > Global shortcuts.

Ctrl + Alt + Shift + F7
Toggles between two predefined keyboard layouts. You can define these layouts at Main menu > Preferences > Keyboard or by selecting one from "Installed keyboard layouts" menu.
Ctrl + Alt + Shift + F8
Toggles "Never display OSD" also known as the "Private mode" option. Useful when you type passwords, but you do not want the entire functionality of KeyPress OSD disabled. If the Clipboard History Manager is enabled, it will no longer record what you copy to the clipboard. This is to ensure better privacy for you.
Ctrl + Alt + Shift + F9
Toggles between two OSD positions. You can define these at Main menu > Preferences > OSD appearances.
Ctrl + Alt + Shift + F10
Toggles Silent mode.
Ctrl + Alt + Shift + F11
Enables the automatic detection of keyboard layouts.
Ctrl + Alt + Shift + F12
Reinitializes/restarts the OSD.
WinKey + Insert *(0)
Captures the entire text from the currently active text field and displays it in the OSD.
WinKey + Alt + Insert *(0)
Captures only the line of text where the caret / cursor is located from the currently active text field and displays it in the OSD.
Ctrl + Shift + Insert *(1)
Pastes the text from the OSD.
Ctrl + Alt + Insert *(1)
Replaces the text from the currently active field with the one from the OSD.
Ctrl + Alt + Capslock
Activates 'Alternate typing mode'. This allows you to type directly into KeyPress and on {Enter}, the text you typed is sent to the host application..
Shift + Pause / Break
Suspends the script and all of its functionality.

(0). These shortcuts are available only if typing mode is not disabled.

(1). These shortcuts are available only if the option "Enable global shortcuts to paste the OSD content into active text field" is activated. If typing mode is not enabled, these shortcuts are disabled.
